# E-Commerce Data Analysis and Machine Learning Project

## Introduction

This project involves analyzing the Olist Brazilian E-commerce dataset to gain insights into customer behavior and product performance. Using Apache Spark for data processing and analysis, and Spark MLlib for machine learning, this project aims to demonstrate the power of big data technologies in extracting valuable insights from e-commerce data.

## Dataset Information

The dataset used in this project is the Olist Brazilian E-commerce dataset, which includes orders, products, reviews, and customer information.

- **Orders Dataset:** Contains information about customer orders.
- **Products Dataset:** Contains product details.
- **Reviews Dataset:** Contains customer reviews.
- **Payments Dataset:** Contains payment information.
